AWS 工具分类

探索AWS提供的各种云服务工具，助力您的云计算项目

Amazon EC2
计算

弹性云计算服务，提供可扩展的计算容量。

使用场景：Web应用托管、批处理、高性能计算、开发测试环境

Amazon EC2是AWS的核心服务之一，允许用户租用虚拟服务器来运行应用程序。它提供完全控制的计算环境，支持多种实例类型以满足不同的计算需求。

主要特点：

  • 多种实例类型（计算优化、内存优化、存储优化等）
  • 按需实例、预留实例和竞价型实例的灵活定价模式
  • 与其他AWS服务的无缝集成
官方文档
AWS Lambda
计算

无服务器计算服务，让您无需管理服务器即可运行代码。

使用场景：事件驱动型应用、后端微服务、数据处理

AWS Lambda让您无需预置或管理服务器即可运行代码。您只需为实际消耗的计算时间付费，代码未运行时不产生费用。

主要特点：

  • 支持多种编程语言（Node.js、Python、Java、Go等）
  • 自动扩展，从每天几个请求到每秒数千个请求
  • 与其他AWS服务的原生集成
官方文档
Amazon ECS
计算

高度可扩展的容器编排服务，支持Docker容器。

使用场景：微服务架构、批处理、应用迁移

Amazon ECS是一项完全托管的容器编排服务，让您可以轻松部署、管理和扩展容器化应用程序。

主要特点：

  • 与AWS服务深度集成（如负载均衡、IAM、VPC）
  • 支持Fargate无服务器计算引擎
  • 内置任务调度和集群管理功能
官方文档
Amazon S3
存储

简单存储服务，提供高度可扩展的对象存储。

使用场景：静态网站托管、数据备份、数据湖、内容分发

Amazon S3是一种对象存储服务，提供行业领先的可扩展性、数据可用性、安全性和性能。它适用于从简单的备份到复杂的大数据分析应用的各种使用场景。

主要特点：

  • 99.999999999%的数据持久性
  • 多种存储类别以优化成本
  • 强大的访问控制和加密功能
官方文档
Amazon EBS
存储

弹性块存储，为EC2实例提供持久性块级存储卷。

使用场景：数据库存储、企业应用、吞吐量密集型应用

Amazon EBS提供在EC2实例生命周期之外持久存在的块级存储卷。每个EBS卷都会自动在其可用区内复制，以防止组件故障导致的数据丢失。

主要特点：

  • 多种卷类型（SSD、HDD）以满足不同性能需求
  • 快照功能用于备份和恢复
  • 加密选项保护敏感数据
官方文档
Amazon S3 Glacier
存储

低成本的云存储服务，用于数据归档和长期备份。
